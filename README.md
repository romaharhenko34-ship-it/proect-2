# Café Lumière — Frontend Project

Элегантный и современный веб-сайт для кофейни, разработанный в качестве демонстрации навыков современной фронтенд-разработки. Проект включает в себя интерактивную главную страницу и страницу полного меню.

## ✨ Особенности проекта

* **Адаптивная верстка:** Сайт полностью оптимизирован под мобильные устройства, планшеты и десктопы (Mobile First подход).
* **Интерактивные элементы:** * Кастомное модальное окно бронирования столика.
    * Динамическое переключение категорий меню.
    * Эффект Glassmorphism на навигационной панели.
* **Анимации и эффекты:** Использование CSS Keyframes для плавного появления контента (`fadeInUp`) и интерактивных кнопок с эффектом `shimmer`.
* **Типографика:** Использование шрифтовой пары *Playfair Display* и *Jost* для создания премиального визуального стиля.

## 🛠 Технологический стек

* **HTML5:** Семантическая структура для SEO и доступности.
* **CSS3 (Modern):**
    * **CSS Grid:** Использован для сложной сетки в секции «Атмосфера».
    * **Flexbox:** Для выравнивания компонентов и навигации.
    * **Custom Properties:** Глобальные переменные для легкой смены цветовой схемы.
* **JavaScript:** Функционал переключения табов и управление модальными окнами.
* **Fonts & Icons:** Google Fonts и Font Awesome 6.5.0.
