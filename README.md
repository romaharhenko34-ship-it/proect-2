# 🖼️ PhotoScape — Pinterest Style Gallery

Современное веб-приложение для поиска визуального вдохновения, построенное на Vite с использованием модульной архитектуры и принципов Pinterest Layout.

## 🚀 Технологический стек (7/7)

Проект реализует следующие ключевые концепции:

1.  Vite — Инструмент сборки для мгновенного обновления (HMR) и быстрой разработки.
2.  ES-Modules — Разделение логики на независимые модули (api.js, ui.js, storage.js и т.д.).
3.  DOM Manipulation — Динамическое создание карточек и рендеринг контента без перезагрузки.
4.  Event Handling — Обработка поиска, переключения вкладок, модальных окон и скролла.
5.  Form Validation — Валидация поисковых запросов (минимум 2 символа, защита от спецсимволов).
6.  Local Storage — Сохранение понравившихся фото в личную коллекцию ("Watchlist").
7.  Async/Await — Асинхронная загрузка данных из открытых источников (Picsum/Unsplash).

---

## 🛠️ Особенности реализации

* Masonry Layout: Использование CSS column-count для создания эффекта "плитки", где фото разной высоты идеально подгоняются друг под друга.
* No-API Required: Проект настроен на работу с публичными изображениями, не требующими регистрации или ключей.
* Debounce Search: Оптимизация поиска — запросы отправляются только после того, как пользователь закончил ввод.
